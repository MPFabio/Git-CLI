#Installez cli github :

# Pour Ubuntu-Debian

Instructions :

  Donne les permissions d'exécution au script :

    chmod +x script_git_ubuntu.sh

  Exécute le script :


    ./script_git_ubuntu.sh


# Pour Windows

Instructions :

    Ouvre PowerShell avec les droits administrateur.
    Exécute le script :

      ./script_git_windows.ps1
